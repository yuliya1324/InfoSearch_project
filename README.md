# InfoSearch_project

Intall requirements

```shell
pip install -r requirements.txt
```

To run project

```shell
streamlit run app.py -- [--args]
```

Arguments:
* dir_model
* matrix_filename_bert
* matrix_filename_bm25
* matrix_filename_tfidf
* answers_filename
* device
* count_vectorizer_filename
* tfidf_vectorizer_filename
